# /youtube-title

從一個影片主題產出 **YouTube 搜尋/建議/首頁會被點開**的標題三件套：

1. **標題（title）** — 決定 CTR，YouTube 最關鍵的單一槓桿
2. **縮圖文案建議（thumbnail copy）** — 標題與縮圖要「互補」不是「重複」
3. **描述開頭兩行（前 ~120 字元）** — 摺疊前可見、影響點擊與搜尋

核心信念：**YouTube 標題不是 SEO 標題。** 部落格標題餵 Google 爬蟲、求相關性；YouTube 標題餵的是「人類滑過時 0.5 秒內要不要點」的衝動，鉤子權重遠高於關鍵字堆疊。寫不出會被點的標題 → 影片角度本身有問題，回去重想角度。

> 這個 command 跟部落格用的 `/title-write` 是**兩套**：規則不同、截斷長度不同、鉤子強度不同。影片用這個，部落格文章用 `/title-write`。

---

## 用法

```
/youtube-title [影片主題或一句話需求]
```

範例：

- `/youtube-title 我用 Claude Code 一天寫完一個全端專案`
- `/youtube-title K8s NodePort 三種 Service 怎麼選 教學`
- `/youtube-title 職訓局講師教你 30 天學會 Spring Boot`

---

## YouTube ↔ 部落格 標題的關鍵差異（不要拿 SEO 規則套）

| 維度 | 部落格 SEO（/title-write） | YouTube（本 command） |
|---|---|---|
| 截斷長度 | Google SERP ~30 全形字 | 桌面 ~60 字元、**手機/首頁約 40 字元就截 → 重點塞前 40** |
| 排名/曝光訊號 | `<title>` 進 Google on-page 相關性 | 標題餵搜尋+建議，但**真正驅動是 CTR × watch time**，標題與縮圖綁在一起算 |
| 鉤子強度 | 適度，clickbait 被 dwell time 懲罰 | **更激進**：好奇缺口 / 數字 / 反差是常態（但仍不能標題吹、內容虛，YouTube 用「平均觀看時長」懲罰得更狠）|
| 尾綴 | 站台 `\| 站名` 自動吃額度 | **無自動尾綴**；但有 `(2026)`、`EP01`、`#shorts` 系列慣例 |
| 關鍵字位置 | 主關鍵字前置餵爬蟲 | 主關鍵字仍要前置（搜尋+建議流量），但**情緒鉤子優先級更高** |
| 縮圖關係 | 無 | **標題與縮圖必須互補**：縮圖喊情緒/結果，標題補上下文/數字，不要兩邊講同一句 |

## 流程（5 步，每步都要落字，不准跳）

### Step 0：抓素材 — 這支影片的「點擊理由」是什麼

寫標題前先逼問三題，答不出回去看影片大綱：

1. **主關鍵字**是什麼？（觀眾會在 YouTube 搜尋框打什麼）→ 進標題前 40 字元
2. 這支影片有沒有**具體數字 / 結果 / 時間**？（一天、30 天、3 個坑、省下 X 小時）→ 數字是縮圖與標題最強的具體性來源
3. 這支的**一句話鉤子**是什麼？（反直覺結果 / 痛點 / 懸念 / 對比）→ 這是縮圖要喊的，標題要補的

> 三題寫成 3 行，當所有候選的原料。

### Step 1：產候選 — 至少 6 個，跨 3 種以上公式

對照 `${CLAUDE_PLUGIN_ROOT}/commands/youtube-title.headline-formulas.md`，**強制跨公式**：

- 至少 1 個**數字/結果式**（「我用 X 一天做完 Y」）
- 至少 1 個**好奇缺口式**（「沒人告訴你 X 其實是…」「我犯了這個錯」）
- 至少 1 個**痛點/對比式**（「別再用 X 了，改用 Y」）
- 其餘自由（how-to / 問句 / 第一人稱實測）

每個候選標註用了哪個公式。**YouTube 額外鼓勵第一人稱**（「我」開頭的實測/踩坑比中性陳述 CTR 高）。

### Step 2：打分 — 逐個對照下方 YouTube 計分表

每個候選逐條跑計分表（滿分 100），列分數 + 一句話為什麼。**禁止憑感覺說「這個比較好」**。

排序，主推取最高分。前兩名差 ≤5 → 兩個都列讓使用者選。**主推 < 85 → 回 Step 1 重產**（YouTube gate 設 85，比部落格 90 略低，因為縮圖能補一部分鉤子）。

### Step 3：縮圖文案 + 描述開頭

**縮圖文案**（3-5 字，與標題互補）：
- 縮圖喊**情緒 / 結果 / 數字**（「一天搞定」「3 個坑」「翻車了」）
- 標題補**上下文 / 主題 / 關鍵字**
- 反例：標題「我一天寫完全端專案」+ 縮圖也寫「一天寫完」→ 浪費，兩邊重複

**描述前兩行**（摺疊前可見，~120 字元）：
- 第一行重申鉤子 + 主關鍵字（搜尋會吃）
- 第二行給觀看理由 / 章節暗示（「附完整流程」「踩過的坑一次講完」）

### Step 4：交付格式

```
主推標題：<title>（XX/100，公式：XXX，手機可見前 40 字：<前40字>）
備選：<title 2>（XX）/ <title 3>（XX）

縮圖文案：<3-5 字>（與標題互補點：<一句>）

描述前兩行：
<第一行>
<第二行>

— 為什麼這樣選（2-3 句）：<爛標題 vs 改寫差異，當教學>
```

## YouTube 標題計分表（滿分 100）

| 項目 | 滿分 | 計分規則 |
|------|:---:|------|
| **手機可見度（重點前 40 字元）** | 20 | 主關鍵字+鉤子都落在前 40 字元給 20；只有其一在前 40 給 12；重點被推到 40 後給 0。**用下方 python 算字元數，半形=1 全形=1 都算 1 個顯示位（YouTube 不分全半形，純字元數截斷）**。總長 < 70 字元。 |
| **鉤子 / 好奇缺口** | 25 | 有反直覺/懸念/強對比/痛點其一給 25；只是中性陳述給 12；純模板（「完整教學」「入門」）給 0。**YouTube 這項權重最高。** |
| **具體性（數字 / 結果 / 時間）** | 20 | 有具體數字或明確結果給 20；只有專名沒數字 12；全抽象 0。非整數/奇數可信度更高。 |
| **主關鍵字前置** | 15 | 主關鍵字在前 40 字內給 15；出現但靠後 8；沒出現 0。 |
| **第一人稱 / 情緒張力** | 10 | 第一人稱實測（我…）或命中情緒詞給 10；中性 5。 |
| **可讀性 + 不 clickbait** | 10 | 念一遍順、不堆關鍵字、標題承諾影片真的有給 10；繞口或標題吹內容虛扣到 0（YouTube 用平均觀看時長懲罰最重）。 |

### 字元數實算

YouTube 按**字元數**截斷（不分全半形），但中文一字仍佔較寬，手機列表保守抓 **40 字元** 為可見上限：

```bash
python3 -c "
s='''我用 Claude Code 一天寫完一個全端專案'''
print(f'總字元 {len(s)} / 前40字: {s[:40]}')
"
```

### 禁用詞 + 不准 clickbait

標題不准出現自我貼標籤的行銷話術，逐字檢查避開：

> 終極、極致、必看、震撼、史上最強、絕對、保證、爆款、神級、無腦、躺賺、秒懂（除非真的一看就懂）、革命性、顛覆

clickbait 紅線：標題承諾的「結果 / 數字 / 反轉」影片裡必須真的有。YouTube 的平均觀看時長(APV)會懲罰標題吹牛，比 Google 的 dwell time 更狠 —— 一次 clickbait 拉低 APV，後續影片建議流量一起掉。

## 計分依據（權重不是拍腦袋）

- **YouTube 官方創作者學院**：縮圖 + 標題是「一組」決定 CTR，建議標題與縮圖傳達互補資訊、不重複。→ 對應「縮圖文案互補」「鉤子 25」。
- **手機佔 YouTube 觀看 >70%**，手機列表標題顯示比桌面短 → 重點前置。→ 對應「手機可見度 20」。
- **CTR × 平均觀看時長** 是 YouTube 推薦演算法核心：標題拉點擊、內容守留存，標題吹牛拉高 CTR 但崩 APV 反被降權。→ 對應「不 clickbait 10」權重。
- 數字/奇數/具體結果提升點擊（同部落格 Backlinko/BuzzFeed 研究，YouTube 縮圖數字效果更明顯）。→ 對應「具體性 20」。

## 反模式（看到自己在做就停）

- ❌ 拿 `/title-write` 的 SEO 規則直接套 YouTube（截斷長度、鉤子強度、尾綴邏輯全不同）
- ❌ 只產 1-2 個候選就定案
- ❌ 6 個候選全是 how-to 變體（跨公式是硬性要求）
- ❌ 標題與縮圖文案講同一句話（浪費，要互補）
- ❌ 打分憑感覺不對表
- ❌ clickbait 標題（拉高 CTR 崩 APV，演算法反噬比 Google 狠）
- ❌ 重點關鍵字/鉤子被推到 40 字元後（手機看不到）

## 跟其他 yc-plugin command 的接口

- 標題定好可直接餵給 `/youtube-upload` 的 `--title`、`/youtube-update` 改既有影片標題
- 連載影片改用 `/youtube-series` 的標題模板（含 `EP{episode:02d}` 變數），本 command 用來設計模板裡那段「副標公式」
- 補救既有低 CTR 影片：先 `/youtube-stats` 拉互動率，挑低 CTR 影片用本 command 重寫標題再 `/youtube-update`
